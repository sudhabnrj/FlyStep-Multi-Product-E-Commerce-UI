# Loop Engineering Rules

This project follows Loop Engineering.

Do NOT attempt to perfect the design in one generation.

Generate Version 1 only.

After generation, stop and wait for observation.

The workflow is:

Generate

↓

Observe

↓

Critique

↓

Refine

↓

Generate Again

Do not skip any step.

Only implement one group of improvements per iteration.
